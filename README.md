Some extra geoms, scales, and themes for
[ggplot](http://ggplot2.org).


## Install 

This is not available from CRAN.

To install the current version from github, use the
**devtools** package,

```r
library("devtools")
install_github(c("whjelmar/theme_citihub"))
```
